# 🎵 Zingen

Uma landing page moderna e responsiva para um aplicativo de karaokê, desenvolvida com foco em design elegante, experiência do usuário e animações suaves.

## ✨ Funcionalidades

* 📱 Layout totalmente responsivo
* 🎨 Interface moderna e intuitiva
* ⚡ Animações suaves e efeitos visuais
* 🖼️ Seções organizadas para melhor experiência do usuário
